Project Description

I had breifly mentioned to my supervisor that I was interested in learning to code. He provided me with an idea that football and soccer wanted to pull an image and have it display on the iPad. This was a learn as I go project because I had no experience in programming.

Project Details

  • I used Swift as the language of choice because it was easier to read and understand the code.
  • The coach's in charge of the image will have a network drive that they can drop the picture to the IT web server to display to the iPad.
  • The app has the ability to refresh the page automatically after being minimize to ensure the most recent image is shown.
  • The app will not only be used for the athletes but also as a marketing technique for other colleges that visit our facilites.
